Diferença entre WEB e Internet

O que é Internet?

É uma rede global de computadores interconectados que se comunicam por meio de um conjuto de protocolos.
Nela são executados diversos serviços incluindo a World Wide Web. É a infraestrutura que permite a troca de dados e informações entre dispositivos em todo o mundo.

O que é WEB?

A WEB é um dos muitos serviços disponíveis na Internet.
Ela é um serviço que interconecta documentos utilizando o protocolo HTTP( Hypertext Transfer Protocol), contendo diversos sites e páginas que podem ser acessadas via URLs ou por hyperlinks nas páginas acessadas.
